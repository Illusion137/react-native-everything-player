# React Native Everything Player

This library was based on [React Native Track Player](https://github.com/doublesymmetry/react-native-track-player). \
This is just a hacky solution for [RNTPvE](https://github.com/Illusion137/RNTPvE) but using nitro modules and with android support.

> [!NOTE] RNTPvE is a git clone of both React Native Track Player and SwiftAudioEx but with modifications for YouTube's SABR playback, equalizer and crossfading.

It has a very similar API to `react-native-track-player` so I'd just use that for documentation, this is really just a published internal libary for [Illusi](https://github.com/Illusion137/Illusi) and [Origin](https://github.com/Illusion137/lib-origin).\
Feel free to use those repositories as a kind of documentation?
